About Us

Transforming Tech Education for Kids

We're on a mission to make programming accessible, fun, and engaging for the next generation of innovators

Our Mission

At Jaystarbliss Dynamic Hub, we believe that every child deserves the opportunity to learn programming and technology in a fun, engaging, and accessible way. We break down complex coding concepts into digestible, interactive lessons that spark creativity and innovation.

Our Vision

From Scratch visual programming to Strudel live coding and cutting-edge AI vibe coding with Claude AI, we provide world-class educational experiences that prepare children for the digital future. We don't just teach code — we nurture problem-solvers, creators, and future tech leaders.

Our Services

Comprehensive Tech Education

Discover our range of innovative programs designed to inspire and educate young minds

Scratch Programming

Visual block-based coding that makes programming intuitive and fun. Kids create games, animations, and interactive stories while learning fundamental coding concepts.

Strudel Live Coding

Create music through code! Our live coding sessions teach algorithmic thinking while students compose and perform electronic music in real-time.

AI Vibe Coding with Claude

Explore the future of programming with AI-assisted coding. Students learn to collaborate with Claude AI to build projects and understand artificial intelligence.

Creative Web Design

Learn HTML, CSS, and JavaScript to build stunning websites. From personal portfolios to interactive web apps, we make web development exciting.

Coding Competitions

Participate in exciting coding challenges and hackathons. Build teamwork skills, compete for prizes, and showcase projects to the community.

Mentorship Programs

One-on-one guidance from experienced instructors. Personalized learning paths tailored to each student's interests, pace, and goals.
